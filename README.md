# quran

Minimal HTTP server serving Quran (and Indonesia Translation) via REST API

## Quran data

The original Quran data is fetched from [this project](https://github.com/alfianyusufabdullah/DatabaseAlquranQ). You can see the file under [datasource](./datasource/) directory.
